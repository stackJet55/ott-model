# OTT Management System

A menu-driven **OTT (Over-The-Top) Management System** developed in **C** that simulates the management of streaming platform data, including TV shows, movies, users, cast members, subscriptions, awards, and viewing history.

The project demonstrates the use of structured programming, file-independent data management, searching, sorting, and user/admin operations using C structures and arrays.

## Features

### Admin
- Add TV Shows
- Add Movies
- Add Users
- Add New Cast Members
- Verify Cast Awards
- View TV Shows Ranked by Viewing Time

### User
- Register as a New User
- Search Media by Cast
- Check Subscription Status
- Search by Production Company & Cast
- View Movie Awards
- Display Cast Awards (Descending Order)
- Categorize Users Based on Membership Duration
- Find Most Viewed Movie by Month
- Search Media by Exact Actor–Actress Combination

## Compilation

Using GCC:

```bash
gcc main.c -o ott
```

Run:

```bash
./ott
```

On Windows:

```bash
gcc main.c -o ott.exe
ott.exe
```
